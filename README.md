📌 1. 게임 개요

주사위를 통해 무기를 획득하고, 웨이브마다 등장하는 적과 보스를 처치하며 생존하는 2D 탑뷰 로그라이크 액션 게임이다.
플레이어는 WASD 이동, 우클릭 조준, 좌클릭 공격을 기반으로 전투를 수행하며, 매 플레이마다 달라지는 무기와 상황에 대응하는 전략적 플레이가 핵심이다.

🎲 2. 핵심 시스템 (진행 구조)
게임 시작 시 주사위를 굴려 무기 랜덤 획득>
웨이브마다 다양한 적이 등장하며 난이도 상승>
맵 내 랜덤 위치에 주사위 오브젝트 생성 → 접촉 시 무기 교체>
동일 무기 중복 획득 시 2단계 강화 무기 적용>
4웨이브 이후 보스 등장 전: 필드 초기화 + 보스 스폰>
보스 처치 후 특정 위치 이동 시 게임 클리어>

👉 핵심 루프
주사위 → 무기 획득 → 웨이브 생존 → 강화 → 보스 전투 → 클리어

⚔️ 3. 전투 요소 (적 & 무기 시스템)
▪ 적 구성
1웨이브: 닭 – 기본 추적형
2웨이브: 개 – 속도↑, 체력↑
3웨이브: 곰 – 고체력 탱커
4웨이브: 치타 – 직선 돌진 공격
보스: 대형 거미
패턴 순환
추적
다수 소환
돌진

▪ 무기 시스템
연발총 → 2단계: 산탄 + 스턴 효과
방망이 → 2단계: 망치 + 회전 속도 기반 추가 피해
헬멧 → 2단계: 돌진 공격 → 지속 + 방향 조절 가능
전기헬멧 → 2단계: 장판 피해 → 범위 증가 + 어그로 제한
플라즈마총 → 2단계: 장판 생성 + 슬로우/돌진 차단
에너지 레이저 → 2단계: 직선 공격 → 중복 시 체력 회복

👉 핵심
모든 무기는 강화(2단계) 시 플레이 방식 변화
무기 선택이 곧 전투 전략 결정 요소로 작용

✅ 최종 정리
랜덤 주사위 기반 무기 선택 로그라이크 구조
웨이브 → 강화 → 보스 구조의 명확한 게임 루프
무기 강화에 따라 플레이 스타일이 바뀌는 전략적 전투 시스템
